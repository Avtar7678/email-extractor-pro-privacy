Privacy Policy

Last updated: April 2026  ·  Email Extractor Pro v1.0.0

📌 Short version: We collect nothing. All processing happens locally in your browser. No data is ever sent to any server we operate.

1. What This Extension Does

Email Extractor Pro is a browser extension that scans web pages to find and collect email addresses. It operates in three modes:

  • Quick Scan — reads the content of the currently active tab.
  • Deep Scan — reads the current tab and fetches linked pages on the same site to look for emails.
  • Bulk Scan — fetches a list of URLs provided by the user from an uploaded file and extracts emails from each.

2. Data We Collect

We do not collect, transmit, or store any personal data on external servers. Specifically:

  • The extension reads web page content (HTML source) solely to identify email address patterns.
  • Any email addresses found are displayed only within the extension popup and are never sent anywhere outside your browser.
  • URL lists uploaded via the Bulk Scan feature are processed entirely within your browser's memory and are never uploaded to any server.
  • Exported CSV files are saved directly to your local device using the browser's download mechanism.

3. Permissions Explained

  • activeTab — required to read the content of the page you are currently viewing when you initiate a scan.
  • scripting — required to execute the email-extraction script on the active tab's page.
  • storage — used to save your scan preferences (chunk size, delay settings) locally on your device.
  • Host permission (<all_urls>) — required so the Deep Scan and Bulk Scan features can fetch pages from any website to extract emails. No data from these requests is stored or transmitted beyond your local browser session.

4. Third-Party Requests

During a Deep Scan or Bulk Scan, the extension makes direct HTTP requests to websites you instruct it to scan. These are standard web requests made from your browser — no intermediary server is involved. The extension does not load any third-party scripts, analytics, or tracking libraries.

5. Local Storage

The extension uses chrome.storage.local exclusively to persist user preferences (such as chunk size and delay settings). This data never leaves your device.

6. Children's Privacy

This extension is not directed at children under 13 and does not knowingly collect information from children.

7. Changes to This Policy

If we update this policy, the revised version will be posted at this page with an updated date. Continued use of the extension after changes constitutes acceptance of the new policy.

8. Contact

If you have any questions about this privacy policy, please open an issue on the extension's support page or contact us via the Chrome Web Store listing.
